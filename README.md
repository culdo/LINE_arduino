# LINE_arduino
利用樹梅派收發LINE訊息，並透過Serial控制mbot行走並回傳距離感測器數值。
<img src="docs/demo1.gif" width="80%"/>
<img src="docs/demo2.gif" width="80%"/>
## Requirments
### Hardware
* Raspberry pi 3b+
* Mbot
### Software
#### Arduino 
* [Makeblock Library](https://github.com/Makeblock-official/Makeblock-Libraries)
#### Raspberry Pi Python library
* PySerial
* Selenium
